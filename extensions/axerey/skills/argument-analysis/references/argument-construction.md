# Argument Construction

Build, analyze, and evaluate arguments using memory context and reasoning tools.

## Usage
Use this command when you need to:
- Construct strong arguments from evidence
- Analyze and evaluate existing arguments
- Build balanced perspectives on complex topics
- Develop persuasive reasoning
- Create comprehensive argument frameworks

## Argument Construction Process

### 1. Evidence Gathering
```
1. Use context broker to retrieve relevant memories
2. Use memory search for related knowledge and patterns
3. Get decision patterns from past outcomes
4. Gather supporting data and precedents
5. Organize evidence by strength and relevance
```

### 2. Argument Building
```
1. Identify core claims and premises
2. Structure logical argument flow
3. Use steelman to strengthen opposing viewpoints
4. Use strawman to identify weaknesses
5. Build comprehensive argument framework
```

### 3. Analysis & Evaluation
```
1. Use steelman to strengthen arguments objectively
2. Use strawman to identify distortions and fallacies
3. Use argument pipeline to transform weak arguments
4. Evaluate evidence quality and relevance
5. Check logical consistency
```

### 4. Refinement & Learning
```
1. Strengthen weak points in arguments
2. Address identified fallacies
3. Store successful argument patterns
4. Extract rules from successful arguments
5. Learn from argument effectiveness
```

## Argument Construction Parameters

### Core Parameters
- **argument_type**: technical, business, strategic, policy, ethical
- **claim**: The main argument or position
- **context**: Background and scope for argument
- **evidence_tags**: Memory tags for evidence gathering
- **opposing_view**: Alternative perspective to address

### Analysis Parameters
- **analysis_mode**: steelman, strawman, balanced, comprehensive
- **perspective**: angel, demon, blend, probabilistic
- **memory_limit**: Number of memories to consider (1-20)
- **confidence_threshold**: Minimum evidence strength
- **learning_mode**: Store argument patterns for future use

## Argument Quality Framework

### Evidence Quality
- **Relevance**: How directly evidence supports claims
- **Strength**: Statistical significance and sample size
- **Recency**: How current the evidence is
- **Source**: Credibility and expertise of sources
- **Consistency**: Agreement across multiple sources

### Logical Structure
- **Premise Strength**: Quality of foundational claims
- **Logical Flow**: Clear progression from premises to conclusion
- **Fallacy Detection**: Identification of logical errors
- **Objection Handling**: Addressing counterarguments
- **Conclusion Validity**: Soundness of final claims

### Persuasiveness Factors
- **Credibility**: Trustworthiness of argument and sources
- **Emotional Appeal**: Appropriate use of emotional elements
- **Practical Relevance**: Applicability to real-world situations
- **Clarity**: Clear communication of ideas
- **Completeness**: Comprehensive coverage of relevant factors
