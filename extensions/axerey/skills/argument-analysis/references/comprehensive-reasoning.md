# Comprehensive Reasoning

Unified reasoning system that orchestrates memory, context, and argument analysis tools for sophisticated decision-making and analysis.

## Usage
Use this command for complex reasoning tasks that require:
- Multi-perspective analysis
- Memory-informed decision making
- Argument construction and evaluation
- Context-aware reasoning
- Systematic problem solving

## Reasoning Workflow

### 1. Context Gathering
```
1. Start reasoning session with clear objective
2. Use context broker to retrieve relevant memories
3. Search for related knowledge and patterns
4. Get decision patterns from past outcomes
5. Set up persona for appropriate analytical perspective
```

### 2. Argument Analysis
```
1. Use steelman to strengthen opposing viewpoints
2. Use strawman to identify distortions and fallacies
3. Use argument pipeline to transform weak arguments
4. Analyze decision patterns and success factors
5. Build comprehensive argument framework
```

### 3. Decision Making
```
1. Use mind-balance for probabilistic decision analysis
2. Apply angel/demon advisory system for perspective
3. Consider memory-informed patterns and outcomes
4. Evaluate options with reasoning tools
5. Generate decision with confidence scoring
```

### 4. Learning Integration
```
1. Store reasoning process and outcomes
2. Label decision outcomes for learning
3. Extract rules from successful reasoning patterns
4. Update procedural knowledge
5. Consolidate reasoning insights
```

## Reasoning Parameters

### Core Parameters
- **task**: decision-making, argument-analysis, problem-solving, strategic-planning
- **query**: The question or problem to reason about
- **context**: Additional context for reasoning
- **memory_tags**: Tags to filter relevant memories
- **reasoning_type**: steelman, strawman, mind-balance, systematic-analysis

### Advanced Parameters
- **persona**: Analytical persona to use
- **perspective_mode**: angel, demon, blend, probabilistic
- **memory_limit**: Number of memories to consider (1-20)
- **confidence_threshold**: Minimum confidence for conclusions
- **learning_mode**: Whether to store reasoning process
