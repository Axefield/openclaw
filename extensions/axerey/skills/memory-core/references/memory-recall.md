# Recall Memories

Retrieve specific memories by query or get recent memories for context.

## Usage
Use this command when you need to:
- Get context for current work
- Recall specific decisions or solutions
- Access recent project information
- Find memories from specific sessions
- Get quick access to pinned memories

## Recall Types
- **Query-based**: Find memories matching specific terms
- **Recent**: Get latest memories chronologically
- **Tagged**: Filter by specific categories
- **Session-specific**: Memories from particular sessions

## Example Usage
```
Recall memories about: "trading strategy"
Limit: 10
Tags: trading, strategy
```

## Parameters
- **query**: Search terms (optional, empty for recent memories)
- **tags**: Filter by specific tags (optional)
- **limit**: Number of results (1-50, default: 8)
- **sessionId**: Recall from specific session (optional)

## Tips
- Leave query empty to get recent memories
- Use tags to narrow down results
- Higher limits for comprehensive context
- Use sessionId for session-specific recall
