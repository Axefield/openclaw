# Core Memory Operations

Centralized tools for fundamental memory management tasks supporting semantic, episodic, and procedural memory types.

## Commands

### /memory-store

Store important information in the memory system with rich metadata.

### /memory-search

Semantic search through stored memories using embeddings.

### /memory-recall

Retrieve specific memories or get recent memories with smart memory ranking.

### /memory-update

Modify existing memory content and regenerate embeddings when needed.

### /memory-pin

Mark important memories for quick access and priority retrieval.

### /memory-forget

Permanently delete memories from the system when no longer needed.
