# Learning Session

Structured learning process using memory consolidation and reflection tools.

## Usage
Use this command when you want to:
- Learn new technologies or concepts
- Consolidate existing knowledge
- Create learning summaries
- Track learning progress
- Build expertise systematically

## Learning Process
1. **Session Start**: Begin with clear learning objectives
2. **Knowledge Gathering**: Collect relevant information
3. **Memory Storage**: Store key concepts and insights
4. **Consolidation**: Group related knowledge
5. **Reflection**: Extract lessons and rules
6. **Session End**: Summarize learning outcomes

## Example Usage
```
Learning session:
Topic: "Advanced TypeScript Patterns"
Goal: "Master advanced TypeScript features for better code quality"
Duration: "2 hours"

1. Record learning plan: "Master advanced TypeScript patterns for code quality"
   - Source: 'plan'
   - Tags: ["learning", "typescript", "patterns", "planning"]

2. Gather knowledge from various sources

3. Record learning steps: "Studied generic constraints and mapped types"
   - Source: 'execution'
   - Tags: ["learning", "typescript", "generics", "execution"]

4. Store key concepts and insights

5. Record learning outcome: "Successfully implemented advanced TypeScript patterns"
   - Source: 'execution'
   - Outcome: success
   - Score: 0.85

6. Record belief: "Advanced TypeScript patterns improve code maintainability and type safety"
   - Belief: true
   - Type: semantic

7. Extract rule: "Use generic constraints to create reusable type-safe components"
   - Type: 'procedural'

8. Mark important: "TypeScript advanced patterns are essential for complex applications"
   - Pinned: true
```

## Learning Framework
- **Objective Setting**: Clear learning goals
- **Record Learning Plan**: Document learning approach and goals
- **Knowledge Collection**: Gather relevant information
- **Record Learning Steps**: Document learning process and insights
- **Memory Organization**: Store and tag knowledge
- **Pattern Recognition**: Identify key patterns
- **Record Learning Outcomes**: Document what was learned and effectiveness
- **Rule Extraction**: Create actionable rules
- **Progress Tracking**: Monitor learning progress

## Benefits
- Structured learning approach
- Knowledge consolidation
- Pattern recognition
- Rule-based learning
- Progress tracking
