# Advanced Memory Management

Tools for sophisticated memory organization, pattern recognition, and knowledge consolidation.

## Commands

### /memory-consolidate

Cluster related memories using K-means clustering for organized knowledge structures.

### /memory-extract-rules

Pattern mine executions to generate IF/THEN procedural rules automatically.

### /memory-reflect

Extract distilled lessons and update procedural rules from accumulated experiences.

### /memory-decay

Automatically clean up low-value memories to maintain system efficiency.

### /memory-pin-set

Create curated memory sets for organized, quick access to related memories.

### /daily-summary

Create comprehensive daily activity summaries with key outcomes and learnings.

### /contextual-recording

Record different types of contextual information using the context broker system.

### /contextual-workflow

Complete workflow for recording and retrieving contextual memory stages.

### /memory-types-context

Comprehensive guide to memory types and their contextual roles.
