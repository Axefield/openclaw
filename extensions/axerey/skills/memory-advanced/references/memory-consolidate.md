# Consolidate Memories

Cluster related memories and produce semantic beliefs using K-means clustering.

## Usage
Use this command when you need to:
- Organize scattered related memories
- Create higher-level insights
- Identify patterns across memories
- Generate semantic beliefs
- Clean up memory organization

## Consolidation Process
1. Groups similar memories using K-means clustering
2. Identifies common themes and patterns
3. Creates semantic beliefs from clustered memories
4. Preserves important individual memories
5. Generates insights from memory groups

## Example Usage
```
Consolidate memories:
Tag: "typescript"
Time window: 7 days
```

## Parameters
- **tag**: Specific tag to consolidate (optional)
- **windowDays**: Time window for consolidation (default: 7)

## Benefits
- Better memory organization
- Pattern identification
- Semantic belief generation
- Reduced memory clutter
- Higher-level insights
