1 • What This Is

blingyat is a comprehensive TypeScript Express + Pug web API for advanced trading strategy analysis with Alpaca Market Data integration. Features interactive Plotly charts, advanced strategy engine (15+ patterns), unified signal system, market health dashboard, and WebSocket support.
