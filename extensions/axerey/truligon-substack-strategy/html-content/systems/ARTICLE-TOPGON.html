topgon (Axefield Luck Strategy v1.0) implements a complete trading workflow from signal scanning to order execution, with strict risk management and performance tracking. The modular architecture with pure functions and MCP-driven execution demonstrates sophisticated systems design.

Production-Ready: Express application, TopstepX MCP server, broker adapters (Alpaca/TopstepX). Unique IP: AlphaAxe Radar, AlphaAxe Router, RiskManager system with per-trade, daily, weekly R guardrails.

Broker Adapters: Alpaca & TopstepX

topgon includes broker adapters for Alpaca and TopstepX:

Adapter Architecture

  • Pure Functions: Strategy code accepts data and returns typed objects—no I/O
  • MCP-Driven Execution: Broker adapters return order objects; MCP decides execution
  • Strongly Typed: Full TypeScript support with strict typing

TopstepX Adapter
