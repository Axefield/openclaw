1 • What This Is

topgon (Axefield Luck Strategy v1.0) is a TypeScript trading strategy system that implements a complete trading workflow from signal scanning to order execution, with strict risk management and performance tracking. Pure functions, MCP-driven execution, strongly-typed, modular architecture.

Core Components: AlphaAxe Radar (scanner, ranking), AlphaAxe Router (alerts, entry plans), RiskManager (per-trade, daily, weekly R guardrails), KPIs tracker, Schedule/Playbook helpers, broker adapters (Alpaca/TopstepX).

2 • Why It's Defensible

  • Unique IP: AlphaAxe Radar, AlphaAxe Router, RiskManager system
  • Modular Architecture: Separate concerns (radar, router, risk, metrics, schedule)
  • Strongly Typed: Full TypeScript support with strict typing throughout
  • MCP-Driven Execution: Broker adapters return order objects; MCP decides execution
  • Pure Functions: Strategy code accepts data and returns typed objects—no I/O
  • Production-Ready: Express application, TopstepX MCP server, broker adapters

3 • Near-Term Path

Current Status: Production-ready trading strategy system with modular architecture, broker adapters, and MCP-driven execution.

  • Immediate: Continue development, enhance AlphaAxe Radar and Router
  • Short-Term: Expand broker adapters, improve RiskManager
  • Medium-Term: B2B API offering, licensing opportunities

4 • Key Features

  • AlphaAxe Radar: Scanner and ranking system (VWAP_REVERSAL, MA150_PULLBACK, GAP_FADE)
  • AlphaAxe Router: Alert generation and entry plan creation
  • RiskManager: Per-trade, daily, weekly R guardrails
  • KPIs Tracker: Performance metrics (expectancy, win rate, latency, A+ selectivity)
