How Agentic Systems Work

The agentic systems execution model combines mythoGon with Axerey:

Role Definition

  • Persona Creation: Each role has its own Axerey persona (e.g., p1-ceo, p2-cto)
  • Memory Isolation: Each role has isolated memory space
  • Custom Tools: Each role has custom MCP tools tailored to responsibilities
  • Decision-Making: mythoGon provides soul/agency for role-specific decisions

Human-Agent Collaboration

  • Human Manager: Human manages the agent in their role
  • Agent Execution: Agent executes tasks with decision-making capability
  • Collaboration: Human and agent collaborate on decisions
  • Oversight: Human provides oversight and guidance
