## Semigon: The Game System

### What Is Semigon?

**Semigon is "truligon-rpg"** - A TypeScript RPG core system based on the Truligon universe.

**Core Features:**
- **52-card structured universe** mapped to zones + gameplay
- **Time logic and fractures system** - Fractal evolution of time
- **Player state, tokens, ranks, and suits**
- **Procedural board generation (PCG)**
- **Expandable abilities and turn-based logic**

**The Game System:**
```typescript
// Core commands
startNewGame(characters) — start a board session
applyFracture(game, characterId) — fractal evolution of time
useAbility(game, userId, ability) — activate a skill
```

**Core Lore Principles:**
- **Board = Deck = Time**
- Zones shift logic based on suit, rank, and card history
- Fractures are progression systems, not punishments

**The Philosophy:**
> "Enter the game. Reshuffle yourself."
