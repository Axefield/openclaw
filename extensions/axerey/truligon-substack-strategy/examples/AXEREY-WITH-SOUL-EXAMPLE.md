## The Integration

### How They Work Together

**1. Memory Informs Decisions**

```typescript
// Axerey provides context
const context = await axerey.context_broker({
  task: "decision-making",
  tags: ["trading", "strategy"],
  k: 5
});

// mythoGon evaluates with context
const decision = await mythoGon.runDecision({
  context: {
    ...context,
    domain: "TRADING",
    description: "Should I execute this trade?"
  },
  contract: {
    coreValues: {
      riskManagement: 0.9,
      profitability: 0.8,
      consistency: 0.7
    }
  }
});
```

**What this means:**
- Axerey provides relevant memories
- mythoGon evaluates through soul layers
- Decision is informed by memory AND values
- Decision has agency, not just generation

**2. Decisions Update Memory**

```typescript
// mythoGon makes a decision
const decision = await mythoGon.runDecision(context);

// Axerey stores the decision and outcome
await axerey.memorize({
  text: `Decision: ${decision.creation.chosenOptionId}`,
  tags: ["decision", "trading"],
  type: "episodic",
  source: "execution",
  features: {
    instinct: decision.core.instinct,
    intuition: decision.sense.intuitionDirection,
    cosmicApproval: decision.creation.cosmicApproval
  }
});
```

**What this means:**
- mythoGon makes decisions with agency
- Axerey learns from decisions
- System improves from both memory and soul
- Continuous improvement with values

**3. Adaptive Learning with Soul**

```typescript
// Axerey tracks outcomes
await axerey.label_outcome({
  execId: decision.id,
  outcome: "success",
  score: 8.5
});

// mythoGon reflects on decision
await mythoGon.registerOutcome(decision.id, {
  outcomeNotes: "Trade executed successfully",
  occurredAt: new Date().toISOString()
});
```

**What this means:**
- Axerey learns from outcomes
- mythoGon reflects on decisions
- Both systems improve
- System becomes better with values
