# Pin Memory

Pin important memories for quick access and priority in search results.

## Usage
Use this command when you want to:
- Mark frequently accessed memories
- Prioritize important decisions or solutions
- Create a quick reference set
- Ensure key information is always available
- Organize critical project knowledge

## When to Pin
- Frequently referenced solutions
- Important project decisions
- Key coding patterns or standards
- Critical configuration details
- Essential troubleshooting steps

## Example Usage
```
Pin memory ID: "mem_12345"
Pinned: true
```

## Parameters
- **id**: Memory ID to pin/unpin (required)
- **pinned**: true to pin, false to unpin (default: true)

## Benefits
- Pinned memories appear first in search results
- Quick access to critical information
- Better organization of important knowledge
- Improved context retrieval
- Reduced search time for key items
