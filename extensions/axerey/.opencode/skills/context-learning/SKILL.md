# Context and Learning

Tools for task-specific memory context, adaptive learning, and outcome tracking.

## Commands

### /context-broker

Get task-specific memory context for planning, execution, or review phases.

### /context-grading

Rate context helpfulness to improve adaptive ranking system over time.

### /outcome-labeling

Label execution outcomes with success metrics for system learning.

### /context-explanation

Get explanations for why specific memories were selected for context.

### /adaptive-learning

Monitor and manage the adaptive learning system for continuous optimization.
