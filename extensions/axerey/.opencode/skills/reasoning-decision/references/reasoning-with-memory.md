# Reasoning with Memory

Perform reasoning tasks using relevant memory context for better decisions.

## Usage
Use this command when you need to:
- Make decisions based on past experience
- Solve problems using historical context
- Analyze arguments with relevant background
- Apply learned patterns to new situations
- Make informed choices

## Reasoning Types
- **decision-making**: Make choices based on past outcomes
- **argument-analysis**: Analyze arguments with historical context
- **problem-solving**: Solve problems using past solutions

## Example Usage
```
Reasoning task: decision-making
Query: "Should we use TypeScript for this new project?"
Memory tags: ["typescript", "project-decisions"]
Reasoning type: mind-balance
Use memory context: true
```

## Parameters
- **task**: decision-making, argument-analysis, or problem-solving (required)
- **query**: The question or problem to reason about (required)
- **memoryTags**: Tags to filter relevant memories (optional)
- **reasoningType**: mind-balance, steelman, strawman, or pipeline (optional)
- **useMemoryContext**: Whether to use memory context (default: true)
- **memoryLimit**: Number of memories to consider (1-20, default: 5)

## Benefits
- Context-aware reasoning
- Learning from past experience
- Better decision making
- Pattern recognition
- Informed problem solving
