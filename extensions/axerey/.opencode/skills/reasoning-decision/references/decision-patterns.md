# Decision Patterns

Analyze decision patterns and extract insights from stored outcomes.

## Usage
Use this command when you want to:
- Understand decision-making patterns
- Identify successful strategies
- Learn from past decisions
- Improve future choices
- Analyze decision effectiveness

## Pattern Types
- **success**: Patterns that lead to successful outcomes
- **failure**: Patterns that lead to failures
- **confidence**: Confidence levels and accuracy
- **topics**: Decision patterns by topic area

## Example Usage
```
Analyze decision patterns:
Time window: 30 days
Min occurrences: 3
Pattern type: success
```

## Parameters
- **timeWindow**: Days to analyze (default: 30)
- **minOccurrences**: Minimum pattern occurrences (default: 3)
- **patternType**: success, failure, confidence, or topics (default: success)

## Benefits
- Pattern recognition
- Learning from experience
- Improved decision making
- Success strategy identification
- Failure prevention
