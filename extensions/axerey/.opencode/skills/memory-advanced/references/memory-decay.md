# Memory Decay

Maintain memory scores and delete/archive stale items automatically.

## Usage
Use this command when you need to:
- Clean up low-value memories
- Maintain memory quality
- Remove outdated information
- Optimize memory storage
- Prevent memory bloat

## Decay Process
1. Evaluates memory importance and usage
2. Reduces scores for unused memories
3. Archives or deletes low-value items
4. Maintains high-quality memory set
5. Optimizes storage efficiency

## Example Usage
```
Run memory decay:
Max items per tag: 100
Min importance: 0.3
Decay half-life: 30 days
```

## Parameters
- **maxItemsPerTag**: Maximum memories per tag (optional)
- **minImportance**: Minimum importance threshold (0-1, optional)
- **decayHalfLifeDays**: Decay rate in days (optional)

## Benefits
- Automatic memory cleanup
- Quality maintenance
- Storage optimization
- Focus on important memories
- Reduced noise
