# Daily Summary

Create a comprehensive summary of daily activities, outcomes, and key insights.

## Usage
Use this command when you want to:
- Review daily accomplishments
- Capture key activities and outcomes
- Identify patterns in daily work
- Track progress over time
- Create daily reports

## Summary Contents
- Key activities performed
- Important outcomes achieved
- Rule breaches or deviations
- New beliefs or insights formed
- Progress toward goals

## Example Usage
```
Create daily summary:
Date: 2024-01-15
Tags: ["development", "typescript", "backend"]
```

## Parameters
- **date**: Specific date for summary (optional, defaults to today)
- **tags**: Filter by specific tags (optional)

## Benefits
- Daily progress tracking
- Pattern identification
- Accomplishment documentation
- Insight capture
- Progress monitoring
