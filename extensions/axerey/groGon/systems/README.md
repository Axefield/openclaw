# GroGon Systems Architecture

This directory contains the backend systems and services for GroGon.

## System Structure

```
systems/
├── api/              # Main API server
├── erp/              # ERP integration
├── crm/              # CRM integration
├── services/         # Service management system
├── manufacturing/    # Manufacturing systems
├── flight-planning/  # Flight planning software
├── data-processing/  # Image/video processing
└── shared/           # Shared utilities and types
```

## Technology Stack

- **Runtime**: Node.js 18+
- **Language**: TypeScript
- **Framework**: Express.js or FastAPI (Python)
- **Database**: PostgreSQL
- **Cache**: Redis
- **Message Queue**: RabbitMQ or AWS SQS

See individual system READMEs for setup instructions.
