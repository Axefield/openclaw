# Next Steps Suggestions

Get AI-suggested next reasoning steps based on current state and context.

## Usage
Use this command when you need to:
- Get guidance on next actions
- Discover open questions
- Identify contradictions to resolve
- Find unverified claims
- Plan reasoning workflow

## Suggestion Types

- **Context Gathering**: Retrieve additional relevant memories
- **Verification**: Verify unverified claims
- **Connection Building**: Create missing connections
- **Quality Evaluation**: Evaluate memory quality
- **Contradiction Resolution**: Resolve contradictory memories
- **Pattern Discovery**: Extract rules from patterns

## Example Usage
```
Get next steps:
Session ID: session-123
Limit: 5
Context: "typescript development"
```

## Parameters
- **sessionId**: Session identifier (optional)
- **limit**: Number of suggestions (1-10, default: 3)
- **context**: Optional context for relevance

## Suggestion Logic

Suggestions are generated by analyzing:
- Recent memories and their status
- Memory connections and gaps
- Unverified claims
- Contradictions
- Open questions
- Quality issues

## Benefits
- **Guidance**: Get actionable next steps
- **Discovery**: Find important tasks you might miss
- **Efficiency**: Optimize reasoning workflow
- **Completeness**: Ensure thorough reasoning
- **Learning**: Learn from suggested patterns

## Use Cases

### Planning
Get suggestions before starting a reasoning session.

### Mid-Session
Get suggestions during reasoning to identify next actions.

### Review
Get suggestions after completing work to find follow-ups.

### Optimization
Get suggestions to improve reasoning quality and completeness.

## Best Practices
- Review suggestions regularly
- Prioritize suggestions by importance
- Use suggestions to guide workflow
- Don't blindly follow all suggestions
- Customize suggestions based on context
