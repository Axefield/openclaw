# Vector Search Status

Check Vector Similarity Search (VSS) status and performance metrics.

## Usage
Use this command when you need to:
- Monitor VSS system health
- Check embedding performance
- Troubleshoot search issues
- Optimize vector operations
- Verify VSS installation

## VSS Status Information
- **Installation Status**: Whether VSS is properly installed
- **Performance Metrics**: Search speed and accuracy
- **Index Statistics**: Vector index size and health
- **Embedding Status**: Current embedding provider status
- **Fallback Status**: Whether using fallback search methods

## Example Usage
```
Check VSS status
Get vector search metrics
Verify embedding performance
```

## VSS Features
- **HNSW Algorithm**: Fast approximate nearest neighbor search
- **SQLite Integration**: Native database integration
- **Metadata Filtering**: Combine vector similarity with SQL filters
- **Performance Monitoring**: Real-time performance metrics
- **Automatic Fallback**: Falls back to cosine similarity if needed

## Performance Metrics
- **Search Speed**: Average query response time
- **Index Size**: Number of vectors in index
- **Memory Usage**: RAM usage for vector operations
- **Accuracy**: Search result quality metrics
- **Throughput**: Queries per second capacity

## Benefits
- **Performance Monitoring**: Track VSS system health
- **Optimization**: Identify performance bottlenecks
- **Troubleshooting**: Diagnose search issues
- **Capacity Planning**: Understand system limits
- **Quality Assurance**: Ensure search accuracy

## Best Practices
- Regular VSS status checks
- Monitor performance metrics
- Address performance issues promptly
- Use VSS for large memory datasets
- Fall back to cosine similarity when needed
