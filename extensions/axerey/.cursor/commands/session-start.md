# Start Session

Begin a new working session with specific goals and context tracking.

## Usage
Use this command when you want to:
- Start a focused work session
- Track progress on specific tasks
- Organize memories by session
- Set clear objectives
- Enable session-specific memory retrieval

## Session Types
- **general**: General development work
- **reasoning**: Complex problem-solving tasks
- **decision-making**: Important decision processes
- **argument-analysis**: Analyzing arguments or debates

## Example Usage
```
Start session:
Goal: "Implement user authentication system"
Tags: ["auth", "security", "backend"]
Session Type: general
```

## Parameters
- **goal**: Clear description of session objective (required)
- **tags**: Array of relevant tags (optional)
- **sessionType**: general, reasoning, decision-making, or argument-analysis (default: general)

## Benefits
- Organized memory tracking
- Clear session objectives
- Better context retrieval
- Progress monitoring
- Session-specific insights
