# End Session

Conclude a working session with summary and review.

## Usage
Use this command when you want to:
- Wrap up a work session
- Create session summary
- Review accomplishments
- Capture lessons learned
- Close session context

## Example Usage
```
End session:
Session ID: "sess_12345"
Summary: "Successfully implemented JWT authentication with proper error handling and validation. Created user registration and login endpoints with password hashing."
```

## Parameters
- **sessionId**: ID of session to end (required)
- **summary**: Brief summary of session accomplishments (optional)

## Benefits
- Session closure and organization
- Summary of accomplishments
- Lessons learned capture
- Context cleanup
- Progress tracking
