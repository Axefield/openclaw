# Performance Metrics

Get adaptive ranker performance metrics and system status.

## Usage
Use this command when you need to:
- Monitor system performance
- Check memory retrieval effectiveness
- Evaluate adaptive learning progress
- Troubleshoot performance issues
- Optimize system settings

## Metrics Included
- Adaptive ranker performance
- Vector search status
- Memory retrieval accuracy
- System efficiency metrics
- Learning progress indicators

## Example Usage
```
Get performance metrics
```

## Parameters
No parameters required - provides comprehensive system metrics.

## Benefits
- System health monitoring
- Performance optimization
- Learning progress tracking
- Issue identification
- Data-driven improvements
