# Forget Memory

Delete a memory from the system permanently.

## Usage
Use this command when you need to:
- Remove outdated or incorrect information
- Clean up duplicate memories
- Delete temporary or test memories
- Remove sensitive information
- Prune low-value memories

## When to Forget
- Information is no longer relevant
- Memory contains errors that can't be corrected
- Duplicate of another memory
- Temporary or test data
- Sensitive information that shouldn't be stored

## Example Usage
```
Forget memory ID: "mem_12345"
```

## Parameters
- **id**: Memory ID to delete (required)

## Important Notes
- This action is permanent and cannot be undone
- Consider updating instead of forgetting if information is partially correct
- Use decay function for automatic cleanup of low-value memories
- Be cautious with important project information
